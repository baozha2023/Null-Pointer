# 三章难度与内容对比

本文档展示第一章到第三章的敌人阵容、数值、机制和在不同难度下的变化。

---

## 6. 数值对比总表

### 6.1 普通战斗平均伤害/回合（D=0）

| | 第一章 | 第二章 | 第三章 |
|---|---|---|---|
| 单敌人最小 | 3-5 (沙箱探针) | 5 (火墙兵攻击) | 8×2 = 16 (超频核心) |

| 状态威胁 | 极少 | 脆弱、虚弱、尖刺 | 过热、脆弱+虚弱双 debuff、自增益 |

### 6.2 精英血量/回合伤害对比（D=0）

| | 第一章 | 第二章 | 第三章 |
|---|---|---|---|
| 最低血量 | 45-55 (竞态鬼) | 65-75 (验证者) | 85-95 (炸弹兵) |
| 最高血量 | 100 (递归妖) | 110 (风暴王) | 130 (暴击兽) |

| | 守护兽 (Ch1) | 火墙守将 (Ch2) | 核霸主 (Ch3) |
|---|---|---|---|
| 血量 D=0 | 200 | 230 | 300 |

### 6.4 各难度下三章敌人强化矩阵

```
难度 0：三章均为基准值
    第一章：100/200 血量区间的敌人
    第二章：100/210 血量区间的敌人（+0%~+5%）
    第三章：115/260 血量区间的敌人（+15%~+30%）

难度 1：全局敌人+普通敌人强化
    第一章普通战变难（全局 enemies 1-4 升级）
    第二章普通战变难（普通敌人升级）
    第三章普通战变难（普通敌人升级）

难度 2：精英敌人强化
    第一章精英更强（递归妖/竞态鬼升级）
    第二章精英更强（风暴王/验证者升级）
    第三章精英更强（暴击兽/炸弹兵升级）

难度 3：Boss 强化
    第一章 Boss 守护兽血量+伤害升级
    第二章 Boss 火墙守将血量+护盾+伤害升级
    第三章 Boss 核霸主血量+护盾+伤害升级
```

## 7. 代码架构总结

```
autoload/
├── GlobalProdDataGenerator.gd     # 主入口，委托调用
│
├── act/
│   ├── global_enemies.gd          # 全局通用敌人 (4个: enemy_1~4)
│   ├── act_one.gd                 # 第一章 (5个专属 + 事件池 + ActData)
│   ├── act_two.gd                 # 第二章 (7个专属 + 事件池 + ActData)
│   └── act_three.gd               # 第三章 (7个专属 + 事件池 + ActData)
│
├── card_generators/               # 卡牌定义
│   ├── green_cards.gd
│   └── red_cards.gd
│
└── ...                            # 其他 autoload 文件
```

每个 act 文件独立管理自己的敌人、事件和章节数据，通过 `GlobalProdDataGenerator` 的 `add_enemies()` / `add_events()` / `add_acts()` 统一调用注册。
